# 📱 Sales & HR Management Mobile App

A Flutter-based mobile application designed for **Sales Representatives** and **HR Management**.

This application allows salespersons to manage customer visits, tours, and order bookings, while also providing HR functionalities such as leave management, attendance tracking, expense claims, and location-based punch in/out.

---

## 🚀 Features

## 🧑‍💼 Sales Module

- 📍 Create and manage customer visits
- 🗺️ Plan and manage sales tours
- 🛒 Order booking with product selection
- 📊 View visit history
- 📌 Location-based visit tracking
- 📈 Sales performance tracking

---

## 🏢 HR Module

- 📝 Apply for leave
- 📋 Leave approval workflow
- 💰 Submit and manage expense claims
- ⏱️ Attendance management
- 📍 Location-based punch in / punch out
- 🗺️ Real-time GPS tracking
- 📅 Attendance history

---

## 🛠️ Tech Stack

- Flutter (Latest Stable Version)
- Dart
- REST API Integration
- Provider (State Management)
- Geolocator (Location Services)
- Permission Handler
- SharedPreferences / Hive (Local Storage)
